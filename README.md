# ML BMSTU — Лачина Екатерина Андреевна, ИУ5-63Б

Репозиторий с лабораторными работами и рубежным контролем по курсу **«Технологии машинного обучения»**, МГТУ им. Баумана.

## Структура

| Папка | Тема |
|---|---|
| `lab1_eda` | Разведочный анализ данных. Исследование и визуализация |
| `lab2_preprocessing` | Обработка пропусков, кодирование категориальных признаков, масштабирование |
| `lab3_knn_cv` | Подготовка выборок, кросс-валидация, подбор гиперпараметров (KNN) |
| `lab4_linear_svm_trees` | Линейные модели, SVM, деревья решений |
| `lab5_ensembles` | Ансамбли моделей (бэггинг, бустинг); ноутбук + `theory_lab5_ensembles.txt` |
| `lab6_optimization` | Численная оптимизация, поиск экстремума, визуализация поверхностей |
| `lab7_MLP` | Стекинг, многослойный персептрон (MLP), МГУА; регрессия потребления электроэнергии |
| `RK1_missing_values` | Рубежный контроль 1 — обработка пропусков (вариант 10) |
| `RK2_decision_tree_rf` | Рубежный контроль 2 — дерево решений и случайный лес (вариант 10) |
| `RK2_concept_map` | Рубежный контроль 2 — концептуальная карта (задание, сущности, сборка `dist/`) |

## Стек

`Python` · `pandas` · `numpy` · `scikit-learn` · `matplotlib` · `seaborn`

Дополнительно: `gmdh` (МГУА, лаб. 7, опционально)
